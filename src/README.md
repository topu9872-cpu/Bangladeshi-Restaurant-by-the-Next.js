# 🥘 Banglai Foods | The Essence of Bengal

**Banglai Foods** is a specialized platform dedicated to the rich culinary heritage of Bangladesh. From the spicy curries of the south to the sweet delicacies of the north, this project brings the authentic "Desi" dining experience into the digital age.

---

## ✨ Key Features

- **Heritage Menu:** A deep dive into traditional categories: _Bhorta-Bhaji_, _Pitha-Puli_, and _Biryani_.
- **Spice Level Customization:** A unique feature allowing users to choose their spice tolerance (Mild, Desi-Spicy, or Extreme).
- **Recipe Blogs:** Integrated blog section explaining the history and origins of famous Bengali dishes.
- **Table Booking:** Real-time reservation system for the authentic dine-in experience.
- **Admin Dashboard:** Management tools to update seasonal menus (e.g., Winter Pitha festivals).

## 🚀 Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone: `https://github.com/topu9872-cpu/Bangladeshi-Restaurant-by-the-Next.js.git`
    LIVE Link: `https://bangladeshi-restaurant-by-the-next.vercel.app`

    ```

2.  **Setup Environment Variables:**
    Create a `.env` file and add your database URI and API keys:

    ```env
    DB_URL=your_mongodb_uri
    JWT_SECRET=your_secret_key
    ```

3.  **Run the application:**
    ```bash
    npm run dev
    ```

## 🍱 Signature Banglai Menu Items

> **"Bengali food is not just a meal; it's an emotion."**

- **Sorshe Ilish:** Hilsa fish cooked in a pungent, creamy mustard gravy.
- **Beef Chui Jhal:** A famous spicy beef curry from Khulna cooked with Piper chaba stems.
- **Bhorta Platter:** An assortment of mashed items (Potato, Dried Fish, Eggplant) served with steaming hot rice.
- **Kacchi Biryani:** Raw marinated mutton cooked in layers with aromatic Basmati rice and potatoes.
- **Patishapta Pitha:** Traditional rice-flour crepes filled with jaggery and coconut/kheer.
